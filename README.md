DASHBOARD
=======================================

Dashboard template for boostrap. 

Features
--------
   * Minimize panels.
   * Refresh panels.
   * Move panels.
   * Real-Time graphs.

Requirements
------------
- [Bootstrap](http://getbootstrap.com/)
- [Epoch](https://github.com/fastly/epoch)
- [D3.js](https://github.com/mbostock/d3)
